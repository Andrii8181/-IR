SAD — Статистичний Аналіз Даних 2025

SAD — Статистичний Аналіз Даних 2025 — універсальний калькулятор дисперсійного аналізу для наукових досліджень у садівництві та аграрії. Програма дозволяє проводити одно-, дво- та трифакторний аналіз, розраховувати НІР₀₅ (LSD), перевіряти дані на нормальність за тестом Shapiro-Wilk, а також зберігати результати у TXT та копіювати їх у буфер обміну. Програма має графічний інтерфейс українською мовою, підтримує вставку даних з Excel і зручну роботу з повторностями та групами. Програма SAD розроблена для науковців і агрономів, які проводять експерименти у садівництві та дослідження методів регулювання росту плодових дерев. Вона дозволяє швидко обробляти експериментальні дані та отримувати статистично обґрунтовані результати для наукових публікацій і звітів.

Програма надає такі функції: однофакторний ANOVA з розрахунком НІР₀₅ і CV, дво- та трифакторний ANOVA, перевірку нормальності повторностей за тестом Shapiro-Wilk, збереження результатів у файл TXT, копіювання результатів у буфер обміну, вставку даних з Excel (Ctrl+V або завантаження файлу), дружній інтерфейс для введення, редагування та перегляду даних, кнопки для перегляду опису програми та інформації про розробника.

Вимоги до системи: Python 3.9 або вище; бібліотеки pandas, numpy, scipy, tksheet; Tkinter (входить у стандартну бібліотеку Python). Щоб запустити програму, клонувати репозиторій: `git clone https://github.com/username/SAD_StatAnaliz.git`, встановити залежності: `pip install -r requirements.txt` та запустити: `python nir_master.py`. Після запуску вставляти дані з Excel (Ctrl+C → Ctrl+V) або завантажувати Excel-файли через кнопку "Завантажити з Excel". Натиснути "Аналіз даних" для отримання результатів та використовувати кнопки "Опис програми" та "Про розробника" для перегляду інформації.

Про програму: SAD — Статистичний Аналіз Даних 2025 — це зручний інструмент для швидкого і точного оброблення експериментальних даних, розрахунку дисперсійного аналізу, LSD та перевірки нормальності даних. Програма підходить для дослідників садівництва, агрономії та біології, які працюють з багатофакторними експериментами.
