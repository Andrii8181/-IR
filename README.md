# SAD — Статистичний Аналіз Даних

Програма **SAD (Statystychnyi Analiz Danykh)** — універсальний калькулятор дисперсійного аналізу для одно-, дво- та трифакторних досліджень з перевіркою нормальності Шапіро-Вілка. Підтримує вставку даних вручну, з Excel та копіювання/вставку з буфера.

## Особливості

- Одно-, дво- та трифакторний дисперсійний аналіз
- Розрахунок середніх, дисперсій, F-критеріїв, НІР₀₅
- Перевірка даних на нормальність (тест Шапіро-Вілка)
- Графічний інтерфейс українською
- Можливість додавати рядки та стовпчики, вводити дані вручну, вставляти з Excel
- Збереження та копіювання звітів
